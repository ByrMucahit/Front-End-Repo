Using a ch unit to size text

The ch unit allows you to contol the size of text based on its actual contextual size-the width of a 0 character. This means that if you set the max width of some text as 70ch: regardless of the size of that text, you can always be sure that it will be at most, 70 characters wide.

Go ahead and change the font sizes and max widths of these elements in the CSS editor to see it working.
